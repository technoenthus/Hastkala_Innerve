# HASTAKALA : AI-Powered Marketplace for Indian Artisans

## Project Overview

**Mission:** Connect India's 7 million artisans with global buyers using AI tools that eliminate digital barriers.

**Tagline:** "Where hands remember what machines forget."


---

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Set Up Gemini API Key
- Get free API key: https://aistudio.google.com/app/apikey
- Add to `.env.local`:
```env
GEMINI_API_KEY=your_key_here
```

### 3. Run Development Server
```bash
npm run dev
```

Open **http://localhost:3000** in browser.
